# vitepress-template 
基于vitepress，集成了ts，以及其他简单的配置、主题等，开箱即用，写文档利器

# 基础配置
+  typescript
+  自动注册全局组件
+  eslint 
+  自定义主题

> `ESLint` 默认忽略.开头的目录,  在`.eslintignore`中设置 `!docs/.vitepress` 不让忽略`.vitepress`
